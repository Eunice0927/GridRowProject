# GridRowProject
**GridRow 활용 연습을 위한 숫자 카드 클릭 게임을 만들었다.**

## 게임 설명
- 1~20까지 순서대로 카드를 누른다.
- 알맞은 카드를 누르면 카드가 사라진다.
- 성공, 실패에 따라 결과 화면이 보여진다.
